Example One - Async Code

Javascript is an Asynchronous language, meaning it will not wait to perform actions in order.

If we want to perform a set of instructions in order, we have to use techniques such as callbacks, Promises, Async / Await

You'll be dealing with Asynchronous code when using APIs.

Sometimes it could also be user interactions.

Example Two - Fetch

fetch("https://jsonplaceholder.typicode.com/posts")
.then((response) => { return response.json(); })
.then((json) => { console.log(json) });

The problem with this code is that it uses a lot of nesting. We generally want to avoid this.

Example Three - Async / Await

async function getPosts() {
const response = await fetch('https://jsonplaceholder.typicode.com/posts');
const json = await response.json();
console.log(json);
}
getPosts();

This code looks a lot cleaner, because there is less nesting.
